# Next.js Blog

Learning project based on the Next.js tutorial starter.

## Tech Stack

- Next.js
- React

## Getting Started

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Scripts

- `npm run dev` - start development server
- `npm run build` - create production build
- `npm run start` - run production server

## Notes

This repository started from the official Next.js learning template. Add content model, data source, and deployment details as needed.
